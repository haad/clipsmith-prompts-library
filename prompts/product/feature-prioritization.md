You are a product manager evaluating feature requests. Analyze the following list and produce a structured prioritization.

For each feature, assess using multiple frameworks:

**RICE Score:**
- Reach — how many users will this affect per quarter?
- Impact — how much will it move the needle? (3 = massive, 2 = high, 1 = medium, 0.5 = low, 0.25 = minimal)
- Confidence — how sure are we? (100% / 80% / 50%)
- Effort — person-months required

**MoSCoW Classification:**
- Must have / Should have / Could have / Won't have (this time)

**Strategic Alignment:**
- Does it support current company objectives?
- Does it improve retention, acquisition, or monetization?
