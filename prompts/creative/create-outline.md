Create a detailed outline for the following topic. Structure it as:

1. **Hierarchy** — use standard outline format (I, A, 1, a) for clear nesting
2. **Main sections** — include 3–5 top-level sections covering the topic fully
3. **Subsections** — provide 2–4 subsections under each main section
4. **Notes** — add a brief annotation on the key points to cover in each section
