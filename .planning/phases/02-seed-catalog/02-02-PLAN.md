**Analysis prompts (5 files):**

1. **analyze-data.md** — Title: "Analyze Data", Description: "Extract key insights from data"
   Body: Ask AI to analyze the data and extract key insights. Identify trends, outliers, and patterns. Summarize findings with supporting evidence from the data. Present in order of significance.

2. **compare-options.md** — Title: "Compare Options", Description: "Pros/cons comparison"
   Body: Ask AI to compare the options presented. For each option list pros, cons, and key tradeoffs. Recommend the best option with justification. Use a structured format for easy scanning.

3. **extract-action-items.md** — Title: "Extract Action Items", Description: "Pull action items from meeting notes or text"
   Body: Ask AI to extract all action items from the text. For each item identify: what needs to be done, who is responsible (if mentioned), and any deadline. List in priority order.

4. **identify-risks.md** — Title: "Identify Risks", Description: "Risk analysis of a plan or proposal"
   Body: Ask AI to identify risks in the plan/proposal. For each risk assess likelihood and impact, suggest mitigation strategies. Organize by severity (high/medium/low).

5. **create-summary-table.md** — Title: "Create Summary Table", Description: "Structure text into a comparison table"
   Body: Ask AI to convert the text into a structured Markdown table. Identify the key dimensions for comparison. Include all relevant data points from the source text. Use clear column headers.

**Creative prompts (4 files):**

6. **brainstorm.md** — Title: "Brainstorm Ideas", Description: "Creative brainstorming on a topic"
   Body: Ask AI to brainstorm ideas about the topic. Generate 10+ diverse ideas ranging from conventional to unconventional. For each idea include a one-sentence description. Group related ideas together.

7. **write-story.md** — Title: "Write Story", Description: "Short story from a prompt"
   Body: Ask AI to write a short story based on the prompt/concept. Include vivid characters, a clear setting, and a narrative arc. Keep to 500-800 words. End with a satisfying conclusion.

8. **generate-names.md** — Title: "Generate Names", Description: "Name ideas for projects, products, or features"
   Body: Ask AI to generate name ideas for the described project/product/feature. Provide 15+ options across styles: descriptive, abstract, compound words, and playful. For each name explain the reasoning.

9. **create-outline.md** — Title: "Create Outline", Description: "Structured outline from a topic"
   Body: Ask AI to create a detailed outline for the topic. Use hierarchical structure (I, A, 1, a). Include 3-5 main sections with 2-4 subsections each. Add brief notes on key points to cover in each section.

After creating all 9 files, run `bash scripts/build.sh` to verify they compile.
